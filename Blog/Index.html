O Mundo dos Elementos HTML

Data: 20/01/2024 - 03:45 PM

Utilizada as tags HTML elementos semânticos que são utilizados para estruturar e organizar o conteúdo em uma página web. Eles fornecem informações sobre a função e a estrutura do conteúdo, facilitando a compreensão tanto para os desenvolvedores quanto para os navegadores e motores de busca

header

Função: Representa o cabeçalho da página ou de uma seção específica. Normalmente, inclui elementos como títulos, logotipos, menus de navegação, e outras informações introdutórias.

Exemplo: <header><h1>Meu Site</h1></header>

footer

Função: Representa o rodapé da página ou de uma seção específica. Geralmente, contém informações de copyright, links para páginas relacionadas, dados de contato ou outras informações de finalização.

Exemplo: <footer>&copy; 2024 Meu Site. Todos os direitos reservados.</footer>

nav

Função: Indica uma seção de navegação. É usado para agrupar links de navegação para outras páginas ou recursos dentro do site.

Exemplo: <nav><a href="/">Home</a> <a href="/sobre">Sobre</a></nav>

section

Função: Define uma seção genérica em um documento. Pode ser uma seção temática ou de conteúdo independente.

Exemplo: <section><h2>Seção de Notícias</h2><p>Conteúdo das notícias...</p></section>

article

Função: Representa um conteúdo independente e autônomo, como um artigo, post de blog ou notícia.

Exemplo: <article><h2>Novo Artigo</h2><p>Conteúdo do artigo...</p></article>

aside

Função: Representa um conteúdo que é tangencial ao conteúdo ao redor, muitas vezes usado para barras laterais ou conteúdo relacionado.

Exemplo: <aside><h3>Publicidade</h3><p>Anúncios relacionados...</p></aside>

figure

Função: Utilizado para agrupar qualquer conteúdo que seja referenciado a partir do conteúdo principal, geralmente imagens, gráficos ou vídeos.

Exemplo: <figure><img src="imagem.jpg" alt="Descrição da imagem"><figcaption>Legenda da imagem</figcaption></figure>

figcaption

Função: Fornece uma legenda para o conteúdo dentro de um elemento <figure>.

Exemplo: <figure><img src="imagem.jpg" alt="Descrição da imagem"><figcaption>Legenda da imagem</figcaption></figure>
